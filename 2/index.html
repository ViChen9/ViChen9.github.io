翻篇

说是翻篇，却依然不甘

还是这个名字

启程

时间大概是在三月中旬，拿到了阿里云免费送的云主机使用资格，开始了建站之路。
第一次搭建使用 Apache + php + mysql + WordPress 在只有两兆带宽的阿里云ECS下完成雏形。
网站定名 云煮空楼 ，经一段时间打理之后成果还算满意。

后来在无意之间看到了 Argon 这个主题，被其深深吸引。
便又继续投入，沉迷其中。

荒废

10月3日，收到了服务器已释放的消息，我没有做备份，云煮空楼消散了。
其实早在几天前，就收到过服务器即将到期的消息，但身边却一直有太多事情没处理。
工作的事，学习的事，在那段时间爆发了。

重启

不想做一个有始无终的人，这件事一直搁在心里，麻烦琐事都解决了，重拾这个名字，该继续上路了。
